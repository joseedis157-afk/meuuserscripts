Media Sniffer Advanced

Detector avançado de mídia para páginas web.

O script monitora a página em tempo real e detecta vídeos, áudios e streams carregados por diversos métodos, incluindo elementos HTML, requisições de rede e players populares.

Recursos

✅ Detecção de vídeos e áudios HTML5

✅ Interceptação de Fetch API

✅ Interceptação de XMLHttpRequest (XHR)

✅ Monitoramento dinâmico com MutationObserver

✅ Detecção automática de links de mídia

✅ Suporte a playlists HLS (.m3u8)

✅ Suporte a MPEG-DASH (.mpd)

✅ Leitura de atributos personalizados

✅ Busca de URLs de mídia em scripts inline

✅ Detecção de players populares:

Video.js
HLS.js
DASH.js
JW Player
Plyr
YouTube Embed
Vimeo Embed

✅ Botão flutuante arrastável

✅ Salvamento automático da posição do botão

✅ Reprodutor overlay integrado

✅ Sistema de prevenção de duplicatas

Como funciona

O script utiliza múltiplas técnicas para localizar mídias:

Varredura do DOM
Interceptação de requisições de rede
Monitoramento de alterações em tempo real
Detecção de players conhecidos
Coleta automática de fontes de mídia

Quando uma mídia é detectada, ela é armazenada internamente e pode ser reproduzida através do botão flutuante.
